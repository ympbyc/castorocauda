Spreadsheet Example

Click on cells to edit its expression.
Selecting a cell: COLUMN[ROW]
Builtin Functions: add sub mul div
Example: add(A[1], A[0], 5)
You can lift arbitrary binary a * a -> a functions with spreadsheet.core/lift-tl-fn and use them in the expression.
Example: mod = spreadsheet.core.lift_tl_fn(cljs.core.mod)
Note that self referencing, direct or indirect, will result in a stack overflow.
